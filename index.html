Jan 19 - Sep 19

Real-time object pose estimation using differentiable PnP
Mixed Reality and AI Lab, Microsoft Research, Zürich

  • Created an automated pipeline to generate ground-truth data (binary masks and tight 3D bounding-box vertices) from only RGB images of an object, without any use of depth information or Aruco markers.

  • The pipeline involves 3D reconstruction, point cloud alignment, background subtraction, and morphological operations for extracting camera intrinsic parameters, sparse 3D reconstruction, tight 3D bounding box, object points, binary segmentation masks, and 6D poses for all the images in the dataset.

  • Attempted to improve the accuracy of Microsoft's Single Shot Pose, an approach for simultaneously detecting an object class and its 6D pose, by adding a Perspective-n-Point (PnP) component to the loss function.

High Performance Topology Optimization
Università della Svizzera italiana

  • Achieved 2x speed in solving the Compliance Minimization problem of Topology Optimization by analyzing the performance of different Linear solvers. Moreover, implemented a Linear Elasticity visualizer using FEniCS computing platform.

  • Topology Optimization is a mathematical method that optimizes the layout of material given a set of loads and volume constraints.

  • A Topology Optimization algorithm functions by iteratively redistributing volume across the mesh elements such that compliance is minimized.

  • The computations in this paper were run on a High-Performance Computing Cluster at the Institute of Computational Science.
